# PennyPilot – Personal Finance Tracker

PennyPilot is a simple personal finance tracker built with React. It helps users manage income, expenses, balance, and monthly budget in one place.

## Features
- Add income and expense transactions
- View total income, total expenses, and balance
- Set a monthly budget
- Delete individual transactions
- Clear all transactions
- Data stored using localStorage

## Tech Stack
- React
- JavaScript
- CSS
- Vite

## How to Run
1. Install dependencies
2. Start the development server
3. Open the local host link in the browser
